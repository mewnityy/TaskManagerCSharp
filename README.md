# Task Manager C#

A lightweight commandline task manager built in C#. Add tasks, set priorities and difficulty, track completion - all from the terminal. Data is saved automatically between sessions.

## Data Storage

Tasks are saved automatically to `tasks.json` after every change and loaded on startup.

```json
{
  "Tasks": [
    {
      "Id": 1,
      "Title": "Do some stuff",
      "Description": "upload songs to element social network",
      "Priority": "high",
      "Difficulty": 3,
      "IsDone": false,
      "CreatedAt": "2026-04-18T10:00:00"
    }
  ],
  "NextId": 2
}
```

## Project Structure

```
TaskManagerCSharp/
├── Program.cs        - Entry point, main loop, command routing
├── TaskManager.cs    - Task logic, storage, display, help
├── CommandParser.cs  - Parses raw input, handles quoted strings
└── tasks.json        - Autogenerated data file
```
